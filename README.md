# QR Logo Generator

## What it does
Generates customizable QR codes with your logo embedded in the center. Pure client-side web app - no server needed.

Features:
- Upload logo (PNG/SVG)
- Custom QR data (URL, text, vCard)
- Adjustable logo size/position
- Color schemes
- Download PNG/SVG/PDF

## Who it's for
- Marketers creating campaign QR codes
- Event organizers (tickets, badges)
- Small businesses (menus, business cards)
- Anyone needing branded QR without design software

## How to run
1. Open `index.html` in any modern browser
2. Paste URL/text, upload logo
3. Customize & download
